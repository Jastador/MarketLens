# MarketLens AI

An explainable machine-learning market research dashboard for Indian and international equities.

## Project Goal

MarketLens AI fetches historical market data, engineers technical and statistical features, evaluates machine-learning models, and backtests probability-based market signals.

The project is built for research and education. It does not provide financial advice, guarantee returns, or place trades.

## Planned Features

* Global ticker search
* Historical OHLCV market-data downloads
* Price, volume, volatility, and momentum analysis
* Feature engineering for machine-learning experiments
* Baseline and ML model comparisons
* Walk-forward validation and backtesting
* Interactive Streamlit dashboard
* Explainable signal probabilities and model metrics

## Tech Stack

* Python
* Pandas and NumPy
* yfinance
* Scikit-learn
* Streamlit
* Plotly
* Pytest

## Status

Early development. The first milestone is a reliable ticker-search and historical-data layer.
